Introduction:

iMapInvasives Mobile has been developed to facilitate reporting of invasive species observations from your Android or Apple device. Before beginning you will need to register online with one of the participating states or province at http://www.imapinvasives.org.

Preferences:

“Preferences” are the customized user setting. iMapInvasives Mobile users will need to set their user name and password before proceeding with invasives species observation data collection.

Home Page:

From the home page users can add a new observation record, view records already collected and access all other functionality via the Menu.

Observation Record:

The Observation Record is where a new observation can be recorded or an existing record can be modified prior to upload.
